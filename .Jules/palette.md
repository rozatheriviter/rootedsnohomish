## 2024-05-22 - Accessible Resource Cards **Learning:** Generic link text like "Map" or "Call" in a list is a major accessibility barrier (WCAG 2.4.4). Adding `aria-label` with the resource name (e.g., "Map for [Name]") makes the interface usable for screen reader users without changing the visual design. **Action:** Always ensure repeated action buttons in a list have unique, contextual names via `aria-label`.
